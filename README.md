# Házépítési Kalauz

Független döntéstámogató tudástár házépítőknek. Deno Fresh 2.x + Vite + Preact + Tailwind CSS v4.

### Fejlesztés

```bash
deno task dev       # Fejlesztői szerver (Vite HMR)
deno task build     # Production build
deno task preview   # Production preview
```

### Dokumentáció

A projekt MASTER dokumentációja a `0_MASTER_HUB_DOCS/` mappában található. A fejlesztői instrukciók a `CLAUDE.md`-ben.
